# ci_csharp
Validar Cedula de Identidad Uruguaya (CI)


Para usarlo en tu aplicación c# incluir `CIValidator.cs`



Funciones:
```c#

ci = "1.111.111-1"
CIValidator.Validate(ci);      // => true/false - Valida una cédula completa

```
